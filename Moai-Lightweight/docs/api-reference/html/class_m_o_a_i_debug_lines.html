styleID( number ) See MOAIDebugLines class documentation for a list of styles.

r( number ) Optional. Red component of line color. Default value is 1.
g( number ) Optional. Green component of line color. Default value is 1.
b( number ) Optional. Blue component of line color. Default value is 1.
