MOAIFacebookAndroid Class Reference

Wrapper for Facebook integration on Android devices. Facebook provides social integration for sharing on www.facebook.com. Exposed to lua via MOAIFacebook on all mobile platforms.

ConstantsDescription
MOAIFacebookAndroid.DIALOG_DID_COMPLETE Event code for a successfully completed Facebook dialog.
MOAIFacebookAndroid.DIALOG_DID_NOT_COMPLETE Event code for a failed (or canceled) Facebook dialog.
MOAIFacebookAndroid.SESSION_DID_LOGIN Event code for a successfully completed Facebook login.
MOAIFacebookAndroid.SESSION_DID_NOT_LOGIN Event code for a failed (or canceled) Facebook login.
.
