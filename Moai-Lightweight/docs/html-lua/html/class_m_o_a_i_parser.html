MOAIParser Class Reference

Parses strings using a LALR parser. Generates an abstract syntax tree that may then be traversed in Lua. To use, load a CGT file generated by GOLD Parser Builder. Thanks to Devin Cook: http://www.devincook.com/goldparser.

Inherits MOAILuaObject.
